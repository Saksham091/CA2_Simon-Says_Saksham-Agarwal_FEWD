Instruction

A random colour code will be shown to you. You have to replicate the same code shown to you by clicking on the colour pattern in same order

Note:-

  • Each level will have diffrent rules based on thier difficulty level

      • Follow are the difficulty level and thier rules
    • Noob:- Appearance of colour code will decrease after every 5 levels.
    • Intermediate:- After every level time will increase to 5 sec and code will be shown for lesser time after every 5 level.
    • Pro:- Given time will be lesser than intermediate and code will be shown for lesser time after every level.

  • Wrong code means that the player has lost the game
  • The game is of 10 rounds
  • Every level will give you 10 points, the player must complete all 10 round to win the game
